## 关于我

我对 **AI 如何改善真实业务流程** 很感兴趣，尤其关注电商平台、内容运营、用户沟通、即时通讯场景和智能化自动化。

我在澳大利亚完成了 Information Technology 和 Network Security 相关学业，曾就读于 **Queensland University of Technology** 与 **Griffith University**。过去 5 年，我主要从事 **电商平台内容运营**，现在参与电商平台即时通讯软件接入 AI 的项目，负责 **AI 训练与工作流优化** 相关工作。

我喜欢学习现代 AI 系统如何被训练、评估并应用到真实产品中。除了 AI，我也关注美股市场，喜欢研究优秀科技公司和具备长期价值的资产。

工作之外，我喜欢马拉松和游泳。我曾经从 85kg 减到 65kg，减重 20kg，这件事让我相信长期主义、纪律和持续行动真的会改变一个人。

我也非常喜欢旅行，喜欢在不同城市里感受人文、街道、语言和生活方式。风景不一定总是完美，但重要的是某一个瞬间，我们真的在那里，和喜欢的人一起站在那个地方。我的梦想是环游世界。
