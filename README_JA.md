## 自己紹介

私は **AI が実際のビジネスワークフローをどのように改善できるか** に関心があります。特に、ECプラットフォーム、コンテンツ運営、ユーザーコミュニケーション、インスタントメッセージング、業務自動化に興味があります。

オーストラリアで Information Technology と Network Security を学び、**Queensland University of Technology** と **Griffith University** で学業経験を積みました。過去 5 年間は、主に **ECプラットフォームのコンテンツ運営** に携わってきました。現在は、ECプラットフォームのインスタントメッセージング製品に AI を導入するプロジェクトに参加し、主に **AI トレーニングとワークフロー改善** に取り組んでいます。

現代の AI システムがどのように訓練され、評価され、実際の製品に応用されるかを学ぶことが好きです。また、米国株式市場にも関心があり、長期的な視点で価値あるテクノロジー企業を研究しています。

仕事以外では、マラソンと水泳が好きです。以前、85kg から 65kg まで減量し、20kg の減量に成功しました。この経験から、長期的な考え方、規律、そして継続する力が人を大きく変えることを学びました。

旅行も大好きです。さまざまな街、人々、言語、文化、生活の空気を感じることが好きです。景色が必ずしも完璧である必要はありません。大切なのは、ある一瞬、私たちが本当にそこにいて、好きな人と同じ場所に立っていたということです。私の夢は世界一周です。
